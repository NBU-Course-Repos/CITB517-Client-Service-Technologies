ABOUT THE SITE

This site was create as a project for CITB517 Client-Server Technologies taught in New Bulgarian University (NBU).
The project consists of two parts. The first part , that is currently implemented, is the front-end part of the project. The minimal requirements for the front-end parts are:

  • Title page with an image and description
  • Second page with a gallery of image
  • Comments page, which should display the comments that were previously written

The second part of the project is to implement some sort of back-end functionality, as an addition to the front-end that was developed in the first part. The minimal requirements for the back-end are to be determined.
